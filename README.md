# NotFatherBot 🎉

Телеграм-бот для пар, который превращает критические дни в заботу.

Когда у девушки начинается менструальный цикл — парень получает уведомление
в Telegram с напоминанием "продлить подписку": выполнить одно из желаний 
из её списка. Шоколадка, сериал, обнимашки — всё что она заранее добавила.

## Как это работает

1. Пара регистрируется и создаёт совместный профиль в приложении
2. Девушка заполняет **список желаний** и указывает дату начала цикла
3. Парень получает уведомление в Telegram-бот
4. Парень видит список желаний, выполняет и отмечает выполненное ✓

## Стек

| Слой | Технологии |
|------|-----------|
| Frontend | Nuxt, ShadCN |
| Backend | NestJS |
| Bot | Telegram Bot API |

## Структура проекта

├── frontend/   # Веб-приложение (Nuxt + ShadCN)
└── backend/    # API и Telegram-бот (NestJS)

## Запуск

### Backend
\`\`\`bash
cd backend
npm install
npm run start:dev
\`\`\`

### Frontend
\`\`\`bash
cd frontend
npm install
npm run dev
\`\`\`
